# 三相PWMインバータ シミュレータ — 利用手順書

## 1. 起動方法

### 1.1 desktop UI

```bash
cd 3lvlpwm
pip install -r requirements.txt   # 初回のみ
python main.py
```

起動するとウィンドウが開き、デフォルトパラメータで波形が表示されます。

### 1.2 web UI / API（直接起動）

```bash
cd 3lvlpwm
pip install -r requirements.txt   # 初回のみ
python -m uvicorn webapi.app:app --reload
```

起動後はブラウザで [http://127.0.0.1:8000/](http://127.0.0.1:8000/) を開きます。

- Web UI: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
- API health: [http://127.0.0.1:8000/health](http://127.0.0.1:8000/health)
- OpenAPI docs: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)

web UI は FastAPI 上で配信される静的フロントエンドです。表示単位は desktop UI に合わせて、V_LL は RMS、f_c は kHz、t_d は us、L は mH で入力します。

> **Web UI の外部依存**: 現在の web UI は Plotly を `cdn.plot.ly` から取得します。閉域環境や完全オフライン環境では、Plotly 資産を `webui/` 配下へ同梱する追加対応が必要です。

### 1.3 web UI / API（Docker）

Docker が使える環境では、Python 環境のセットアップなしに 1 コマンドで起動できます。授業用サーバや共有 PC への配布に適しています。

```bash
cd 3lvlpwm
docker compose up --build
```

起動後はブラウザで [http://localhost:8000/](http://localhost:8000/) を開くだけで利用できます。

| コマンド | 内容 |
| --- | --- |
| `docker compose up --build` | イメージビルドと起動（初回・更新時） |
| `docker compose up -d` | バックグラウンド起動 |
| `docker compose down` | 停止 |
| `docker compose logs -f` | リアルタイムログ確認 |

> **ポート変更が必要な場合**: `docker-compose.yml` の `"8000:8000"` を例えば `"8080:8000"` に書き換えて再起動してください。
> **デスクトップ版との共存**: Docker はコンテナ内に web API のみを含みます。同一マシンで `python main.py` による desktop UI を利用することも引き続き可能です。
> **学内サーバ設置時のセキュリティ**: コンテナは `0.0.0.0`（全インターフェース）でリッスンします。インターネット接続サーバへ直接配置する場合は、ファイアウォールまたはリバースプロキシ（nginx 等）で保護することを推奨します。
> **複数同時利用**: デフォルト設定はシングルワーカー（1 プロセス）です。数十名が同時アクセスする学内サーバ環境では `Dockerfile` の CMD 行にある `--workers 1` を実測負荷に合わせて増やすことを検討してください。

### 各パネルの見方

#### ① 変調信号 + キャリア

- **赤**: U相指令 $v_u^*$、**青**: V相指令 $v_v^*$、**緑**: W相指令 $v_w^*$
- **灰**: 三角波キャリア（振幅±1）
- 指令信号がキャリアより大きい区間でスイッチON
- Third Harmonic Injection では、三相へ共通の三次高調波成分が加わり、線間成分を保ったまま線形変調範囲が拡張される

#### ② スイッチングパターン

- 各相のスイッチング信号（0=OFF, 1=ON）を上下にオフセット表示
- 上段: S_u（赤）、中段: S_v（青）、下段: S_w（緑）

#### ③ 線間電圧

- 3 本の線間電圧 $v_{uv}, v_{vw}, v_{wu}$ を step 表示し、PWM の切替タイミングを観察する
- 太線の $v_{uv}$ と破線の基本波を中心に、キャリア周波数に同期したパルス列と包絡線の関係を見る
- 初期状態では $v_{uv}$ のみを表示し、$v_{vw}$ と $v_{wu}$ はトグルで必要時に追加できる

#### ④ 相電圧

- 負荷中性点基準の相電圧 $v_{uN}$ を表示し、局所的なステップと基本波を分けて確認する
- ベースライン設定時は、現在条件とベースラインの相電圧基本波を同じパネルで比較できる

#### ⑤ 相電流

- RL負荷を通過することで、PWMリプルが平滑化されたほぼ正弦波の電流
- 黒破線は、相電圧基本波から計算した理論電流波形
- 情報パネルの「FFT実測」は電流の基本波振幅を表し、理論値と同じ物理量で比較している

#### ⑥ FFTスペクトル

- FFT Signal で **線間電圧 $v_{uv}$** と **相電流 $i_u$** を切り替えられる
- FFT Window で **Hann** と **Rectangular** を切り替えられる
- **青**: 基本波成分（出力周波数 $f$ 付近）
- **赤**: キャリア高調波（$f_c$, $2f_c$, $3f_c$ 付近）
- **灰**: その他の高調波
- 右上に **THD, 基本波振幅, 全RMS値** を表示
- 情報パネルには **V1, I1, THD_V, THD_I, PF1(FFT)** が常時表示される

### 変調率表示（左上）

- `m_a = 0.544` のように現在の変調率を表示
- $m_a > 1$ かつ Overmod View 無効時: **クランプ中表示**
- Overmod View 有効時: **過変調観察中表示**（線形上限超過を警告）

### αβベクトル図の理論背景（空間ベクトル）

空間ベクトル表示では、三相指令 $(v_u, v_v, v_w)$ を Clarke 変換で
$\alpha$-$\beta$ 平面へ写像します。

$$
v_\alpha = \frac{2}{3}\left(v_u - \frac{1}{2}v_v - \frac{1}{2}v_w\right),
\quad
v_\beta = \frac{2}{3}\left(\frac{\sqrt{3}}{2}(v_v - v_w)\right)
$$

平衡三相の正弦指令では、軌跡は理想的に円になります。二相変調（DPWM）は
零相オフセットでクランプ区間を作りますが、零相成分は三相へ共通に加算されるため、
$\alpha$-$\beta$ の基本軌跡そのものは理論上同一です。

過変調観察（Overmod View）では、$m_a > 1$ の領域まで見えるように
Web UI のベクトル図レンジを切り替えています。

- Overmod View OFF: $\pm 1.1$
- Overmod View ON: $\pm 2.2$

また、軌跡が楕円に見えないよう、ベクトル図は等縮尺（aspect ratio 1:1）で描画しています。

## 3. 操作方法

### 3.1 desktop UI

### 変調方式の選択

| 方式 | 内部対応 | 内容 | 観察ポイント |
| --- | --- | --- | --- |
| 三角波比較 | sinusoidal + continuous | 基本の三相正弦波参照を三角波キャリアと比較する | 教科書的な SPWM の基準波形を観察できる |
| 三角波比較(三倍高調波) | third_harmonic + continuous | 零相三倍高調波を注入して線形変調範囲を拡張する | 線間基本波を保ったまま利用率が上がる |
| 三角波比較(二相変調) | sinusoidal + dpwm1 | 三角波比較ベースで 60 度クランプする | スイッチング回数低減とクランプ区間を観察できる |
| 空間ベクトル | minmax + continuous | Min-Max 零相注入による連続 SVPWM 相当 | キャリアベースで空間ベクトル等価の利用率を得られる |
| 空間ベクトル(二相変調) | minmax + dpwm1 | 空間ベクトルベースで 60 度クランプする | 利用率と不連続変調の両方を比較できる |

### FFT 表示切替

| 項目 | 選択肢 | 観察ポイント |
| --- | --- | --- |
| FFT Signal | Line Voltage v_uv / Phase Current i_u | 電圧の高調波群と、RL負荷で平滑化された電流スペクトルを比較できる |
| FFT Window | Hann / Rectangular | 非整数周期や近接高調波の見え方を比較できる。Hann は leakage を抑えやすい |

| スライダー | 表示 | 範囲 | ステップ | 効果 |
| --- | --- | --- | --- | --- |
| V_dc | V_dc [V] | 100–600 V | 1 V | 直流母線電圧。出力電圧のパルス高さが変化 |
| V_LL(rms) | V_LL(rms) [V] | 0–450 V | 1 V | 線間電圧指令のRMS値。変調率 $m_a$ が変化 |
| f | f [Hz] | 1–200 Hz | 1 Hz | 出力周波数。基本波の周期が変化 |
| f_c | f_c [kHz] | 1–20 kHz | 0.1 kHz | キャリア周波数。スイッチング回数が変化 |
| t_d | t_d [us] | 0–10 us | 0.1 us | デッドタイム。零クロス歪みや基本波低下の要因 |
| V_on | V_on [V] | 0–5 V | 0.05 V | 導通経路の固定電圧降下。低電圧時の基本波低下を強調 |
| R | R [Ω] | 0.1–100 Ω | 0.1 Ω | 負荷抵抗。電流振幅が変化 |
| L | L [mH] | 0.1–100 mH | 0.1 mH | 負荷インダクタンス。電流のリプル・位相が変化 |

## 4. 学習シナリオ

### シナリオ1: 変調率の効果を観察する

1. 他のパラメータはデフォルトのまま
2. 変調方式を **三角波比較** に設定する
3. **V_LL(rms)スライダー**を 0V → 200V まで徐々に増加
4. 観察ポイント:
   - ①のパネルで指令信号の振幅が大きくなる
   - ②のパルス幅が広くなる
   - ④の電流振幅が増加する
   - 左上の $m_a$ 値に注目（V_LL=200V で約1.09）
5. 約184V付近（$m_a \approx 1.0$）を超えるとクランプが有効なら**クランプ中**表示
   - これ以上V_LLを上げても出力は変化しない → **線形変調の限界**
6. 理論背景の参照:
   - [αβベクトル図の理論背景（空間ベクトル）](#αβベクトル図の理論背景空間ベクトル)

### シナリオ2: キャリア周波数と高調波の関係

1. デフォルトパラメータで起動
2. **f_cスライダー**を 1kHz → 10kHz まで変化
3. 観察ポイント:
   - ⑥のFFTで、赤いキャリア高調波のピークが右に移動
   - f_cが高いほど**高調波が高周波側に移動→フィルタで除去しやすい**
   - ⑤の電流リプルが減少しやすくなる
   - ⑥のTHDは生のPWM線間電圧に対する指標なので、値の大小だけで良否を判断しない

### シナリオ3: RL負荷の平滑効果

1. R=10Ω, L=0.1mH（最小値付近）に設定
2. ④の電流波形にPWMリプルが大きく現れる
3. **Lスライダー**を 0.1mH → 50mH まで増加
4. 観察ポイント:
   - リプルが急激に減少し、滑らかな正弦波に近づく
   - **インダクタンスの平滑効果**を視覚的に確認
   - 同時に電流の位相遅れが大きくなる（電圧に対して遅れる）

### シナリオ4: 出力周波数と波形品質

1. V_LL(rms)=141V, f_c=5kHzで固定
2. **fスライダー**を 10Hz → 100Hz まで変化
3. 観察ポイント:
   - fが低い → 1周期あたりのスイッチング回数が多い → 波形品質が良い
   - fが高い → スイッチング回数が減る → 電流リプルが増加
   - $m_f = f_c / f$（周波数変調率）が重要な指標

### シナリオ5: 直流母線電圧の影響

1. V_LL(rms)=141V（固定）
2. **V_dcスライダー**を 200V → 600V まで変化
3. 観察ポイント:
   - V_dcが上がると③の線間電圧パルスが高くなるが、基本波成分は変わらない
   - $m_a$ が下がる → パルス幅が狭くなる
   - **同じ出力電圧を得るのにV_dcが高いほど変調率に余裕がある**
4. 理論背景の参照:
   - [αβベクトル図の理論背景（空間ベクトル）](#αβベクトル図の理論背景空間ベクトル)

### シナリオ6: 非理想インバータの影響

1. V_LL(rms)=40V 程度の低電圧条件に設定
2. **t_d** を 0us → 4us、**V_on** を 0V → 1V に増加
3. 観察ポイント:
   - 線間電圧の基本波振幅がわずかに低下する
   - 相電圧の零クロス付近で、理想モデルより歪みが強くなる
   - 相電流波形でも零クロス付近のゆがみが観察しやすくなる
   - 情報パネルの理論値比較は、非理想電圧から計算された基本波に追従する

### シナリオ7: 標準変調方式の比較

1. V_dc=300V, V_LL(rms)=170V, f=50Hz, f_c=5kHz, t_d=0us, V_on=0V に設定
2. 次の変調方式を順に切り替える
   - 三角波比較
   - 三角波比較(三倍高調波)
   - 三角波比較(二相変調)
   - 空間ベクトル
   - 空間ベクトル(二相変調)
3. 観察ポイント:
   - 三角波比較(三倍高調波) と 空間ベクトルでは①の三相指令に零相成分が加わり、③の線間基本波利用率が増加する
   - 二相変調付き 2 方式では②のスイッチング波形に 60 度クランプ区間が現れる
   - ⑥の FFT で方式ごとの高調波分布の違いを比較できる
4. 理論背景の参照:
   - [αβベクトル図の理論背景（空間ベクトル）](#αβベクトル図の理論背景空間ベクトル)

### シナリオ8: 電圧スペクトルと電流スペクトルの比較

1. 変調方式を **三角波比較**、FFT Window を **Hann** に設定する
2. FFT Signal を **Line Voltage v_uv** にして、⑥のキャリア高調波群を確認する
3. FFT Signal を **Phase Current i_u** に切り替える
4. 観察ポイント:
   - 電圧スペクトルではキャリア近傍の高調波が強く見える
   - 電流スペクトルでは RL 負荷の平滑効果で高周波成分が大きく減衰する
   - 情報パネルの **THD_V** と **THD_I** を比較すると、電流側が低くなりやすい
   - **PF1(FFT)** と理論の **cos(φ)** を比較すると、基本波位相差の理解に役立つ

### シナリオ9: SVPWM 二相変調の比較

1. 変調方式を **空間ベクトル** に設定し、基準波形を確認する
2. 変調方式を **空間ベクトル(二相変調)** に切り替える
3. 観察ポイント:
   - スイッチングパターンで 60 度クランプ区間が現れる
   - 線間電圧スペクトルの分布が連続 SVPWM と異なる
   - 低変調率では差が小さく、高変調率で差が見えやすい

## 5. 理論背景

### 変調率

$$m_a = \frac{2 V_{ph,peak}}{V_{dc}} = \frac{2 V_{LL,peak}}{\sqrt{3} \cdot V_{dc}} = \frac{2\sqrt{2} \cdot V_{LL,rms}}{\sqrt{3} \cdot V_{dc}}$$

- $0 \le m_a \le 1$: **線形変調領域**（出力基本波が指令に比例）
- Natural Sampling では $m_a > 1$ でクランプされる
- Third Harmonic Injection では線形上限が $m_a = 2 / \sqrt{3} \approx 1.155$ まで拡張される

### 線間電圧

$$v_{uv} = (S_u - S_v) \cdot V_{dc}$$

3レベル（$+V_{dc}$, $0$, $-V_{dc}$）のパルス電圧。

### 相電圧（負荷中性点基準）

$$v_{xN} = \frac{V_{dc}}{3}(2S_x - S_y - S_z) \quad (x, y, z = u, v, w)$$

### RL負荷の電流応答

$$v_{xN}(t) = R \cdot i_x(t) + L \cdot \frac{di_x(t)}{dt}$$

本シミュレータでは、各サンプル区間で相電圧を零次ホールド入力とみなし、
区分定数入力に対する RL 回路の厳密離散時間解

$$i_{x,n+1} = e^{-R dt / L} i_{x,n} + \frac{v_{x,n}}{R}\left(1 - e^{-R dt / L}\right)$$

を用いて電流を更新する。$R \to 0$ の極限では
$i_{x,n+1} = i_{x,n} + v_{x,n} dt / L$ と一致する。

定常状態での電流振幅の理論値:

$$I_{peak} = \frac{V_{ph,peak}}{\sqrt{R^2 + (2\pi f L)^2}}$$

### THD（全高調波歪み率）

$$\text{THD} = \frac{\sqrt{\sum_{n \neq 1} V_n^2}}{V_1} \times 100\%$$

## 6. 注意事項

- **助走区間**: シミュレーションは自動的に十分な助走区間を計算し、表示は定常状態の最後2周期のみです。パラメータによっては計算に数秒かかることがあります。
- **線形変調の上限**: 三角波比較は $m_a \le 1$、三角波比較(三倍高調波) と空間ベクトルは $m_a \le 2 / \sqrt{3}$ を線形上限とします。Overmod View を有効にするとクランプを無効化して観察できます。
- **変調方式の違い**: 三角波比較は基本 SPWM、三角波比較(三倍高調波) は零相三倍高調波注入、空間ベクトルは Min-Max 零相注入による連続 SVPWM 相当です。
- **サンプリング方式**: 現在は Natural Sampling 固定で、参照波形とキャリアを連続比較します。
- **二相変調の意味**: 名前に「二相変調」が付く方式は DPWM1 ベースの 60 度クランプを用い、連続方式よりスイッチング回数を減らします。
- **理論比較の意味**: 情報パネルの電流比較は、生波形のピーク値ではなく FFT で抽出した基本波振幅を用いています。
- **FFT 表示の意味**: v_uv 表示ではスイッチング起因の高調波分布、i_u 表示では RL 負荷で平滑化された結果を比較できます。
- **窓関数の意味**: Hann 窓は leakage を抑えて非整数周期長でも安定した振幅推定をしやすくします。Rectangular は整数周期条件で直感的なスペクトルを見やすい設定です。
- **RL ソルバの前提**: 電流更新は区分定数の相電圧に対する厳密離散時間解を用いています。PWM ステップ入力に対して RK4 より前提が明確です。
- **デッドタイム/電圧降下モデル**: 簡易非理想モデルとして、デッドタイム中は相電流の向きで導通ダイオードを切り替え、導通経路には固定電圧降下 $V_{on}$ を与えています。
- **非理想モデルの簡略化**: 導通電圧降下は固定値であり、素子のオン抵抗、回復特性、DCリンクリプル、温度依存性は扱っていません。
