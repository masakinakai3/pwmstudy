## モジュール設計（5モジュール + UI）

### ① `reference_generator` — 三相指令信号生成

- **入力**: 線間電圧振幅 V_LL、出力周波数 f、時間配列 t
- **処理**: 線間→相電圧変換 V_ph = V_LL / √3、三相正弦波生成
- **出力**: v_u*(t), v_v*(t), v_w*(t)（正規化された変調信号）

```
v_u* = m_a * sin(2πft)
v_v* = m_a * sin(2πft - 2π/3)
v_w* = m_a * sin(2πft + 2π/3)
```

ここで変調率 m_a = 2 * V_ph / V_dc

### ② `carrier_generator` — 三角波キャリア生成

- **入力**: キャリア周波数 f_c、時間配列 t
- **処理**: 振幅 ±1 の三角波（対称三角波）を生成
- **出力**: v_carrier(t)

### ③ `pwm_comparator` — PWMスイッチングパターン生成

- **入力**: 指令信号3相、キャリア信号
- **処理**: 各相について v_x* > v_carrier なら S_x = 1、それ以外は S_x = 0
- **出力**: スイッチング信号 S_u(t), S_v(t), S_w(t)（0 or 1）

### ④ `inverter_voltage` — インバータ出力電圧演算

- **入力**: スイッチング信号3相、直流母線電圧 V_dc
- **処理**:
  - 相電圧（中性点基準）: v_xN = (V_dc / 3) * (2*S_x - S_y - S_z)
  - 線間電圧: v_uv = (S_u - S_v) * V_dc, 同様に v_vw, v_wu
- **出力**: 相電圧3相、線間電圧3相

### ⑤ `rl_load_solver` — RL負荷電流演算

- **入力**: 相電圧（中性点基準）v_xN(t)、抵抗 R、インダクタンス L、時間刻み Δt
- **処理**: 各相の微分方程式を数値積分（オイラー法 or RK4）

```
v_xN(t) = R * i_x(t) + L * di_x(t)/dt
```

- **出力**: 相電流 i_u(t), i_v(t), i_w(t)

### ⑥ `visualizer` — 波形表示UI

- **表示対象**（サブプロット構成）:
  1. 指令信号 + キャリア信号（重ね描き）
  2. スイッチングパターン S_u, S_v, S_w
  3. 線間出力電圧 v_uv, v_vw, v_wu
  4. 相電流 i_u, i_v, i_w
- **UIコントロール**: スライダーで V_dc, V_LL, f, f_c, R, L を動的変更

## データフロー

```
[ユーザ入力]
  Vll, f, Vdc, fc, R, L
       │
       ▼
  ① 指令信号生成 ──→ vu*(t), vv*(t), vw*(t)
  ② キャリア生成 ──→ vcarrier(t)
       │
       ▼
  ③ PWM比較 ──→ Su(t), Sv(t), Sw(t)
       │
       ▼
  ④ 電圧演算 ──→ vuv(t), vvw(t), vwu(t), vuN(t), vvN(t), vwN(t)
       │
       ▼
  ⑤ RL負荷演算 ──→ iu(t), iv(t), iw(t)
       │
       ▼
  ⑥ 波形表示
```

## 主要パラメータの想定デフォルト値

| パラメータ | 記号 | デフォルト | 範囲 |
|---|---|---|---|
| 直流母線電圧 | V_dc | 300 V | 100–600 V |
| 線間電圧指令（振幅） | V_LL | 200 V | 0–V_dc |
| 出力周波数 | f | 50 Hz | 1–200 Hz |
| キャリア周波数 | f_c | 5 kHz | 1–20 kHz |
| 負荷抵抗 | R | 10 Ω | 0.1–100 Ω |
| 負荷インダクタンス | L | 10 mH | 0.1–100 mH |

## 今後の検討事項

- **シミュレーション時間・時間刻み**: f_c に対して十分な分解能（キャリア1周期あたり100点以上）
- **過変調領域の扱い**: m_a > 1 時の挙動
- **デッドタイム**: 実機を模擬する場合は追加モジュールが必要
- **FFT解析**: 出力電圧・電流のスペクトル表示（学習効果向上）
